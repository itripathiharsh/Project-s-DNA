# Product Roadmap

## Purpose

This roadmap defines the high-level vision and delivery timeline for Project DNA across four major versions. It aligns technical capability with user value, prioritizing local-first analysis for V1 and expanding to team-scale cognition in V2+.

---

## Roadmap Overview

| Version | Theme | Target | Key Capabilities |
|---|---|---|---|---|
| V1 | **Local Cognition** | Individual developer | Single-repo analysis, local LLM, CLI + Tauri UI |
| V2 | **Team Cognition** | Engineering team | Multi-repo, shared SCM, team auth, GraphQL |
| V3 | **Organization Cognition** | Enterprise | Cross-org analysis, plugin system, VSCode extension |
| V4 | **Ecosystem Cognition** | Industry | Ecosystem-wide dependency analysis, CVE correlation, open-source intelligence |

## V1 — Local Cognition

**Goal:** A developer can analyze any codebase on their machine with zero infrastructure.

| Milestone | Features | Delivery |
|---|---|---|
| M1: Foundation | Core SCM, Structural Engine, REST API | Month 3 |
| M2: Reasoning | LLM integration, prompt architecture, reasoning pipeline | Month 5 |
| M3: UI | Dashboard, Evidence Explorer, search | Month 7 |
| M4: Polish | Testing, performance, installation packages | Month 8 |

## V2 — Team Cognition

**Goal:** An engineering team shares a DNA instance for cross-project insights.

| Milestone | Features | Delivery |
|---|---|---|
| M5: Multi-user | PostgreSQL backend, auth, team permissions | Month 10 |
| M6: Collaboration | Shared dashboards, insight comments, notifications | Month 12 |
| M7: Integrations | GitHub/GitLab webhooks, Slack bot, CI/CD integration | Month 14 |

## V3 — Organization Cognition

**Goal:** DNA becomes the cognitive layer for the entire engineering organization.

| Milestone | Features | Delivery |
|---|---|---|
| M8: Scale | Plugin system, custom engines, advanced analytics | Month 18 |
| M9: Ecosystem | VSCode extension, IDE integration, public API | Month 20 |
| M10: Enterprise | SSO, audit logging, on-premise deployment | Month 24 |

## V4 — Ecosystem Cognition

**Goal:** DNA models the external software ecosystem — open-source libraries, SaaS dependencies, and their relationships.

| Milestone | Features | Delivery |
|---|---|---|
| M11: External Data | CVE feed integration, package registry indexing, license scanning | Month 28 |
| M12: Cross-Org | Vulnerability blast radius across dependency chains, ecosystem-wide trend analysis | Month 32 |
| M13: Research | Industry-scale pattern detection, open-source health metrics, academic collaboration | Month 36 |
