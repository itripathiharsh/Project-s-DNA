Architectural API Reference

The DNA Architectural API provides programmatic access to the Knowledge Graph, Analysis Engine, and multi-tenant schema exports. Built for high-frequency synchronization and deterministic resolution.

METHODS

POST /graph/query

Knowledge Graph API

Access the underlying semantic web of dependencies. Query nodes using our GraphQL-lite implementation or perform structural traversals.

QUERY PARAMETERS

depth integer

Traversal recursion limit (Default: 3)

filter string

Node category filtering predicate

REQUEST EXAMPLE content_copy 
curl -X POST "https://api.dna.v2/graph/query" \
  -H "Authorization: Bearer $DNA_KEY" \
  -d '{
    "traverse": "dependency",
    "origin": "node_7721",
    "recursive": true
  }'

Analysis Engine SDK

Leverage the local analysis engine for real-time risk assessment without data exfiltration.

Note: SDK requires DNA Local Runtime v1.2+

TYPESCRIPT USAGE check_circle 
import { DNAEngine } from '@dna/sdk';

const engine = new DNAEngine({
  apiKey: process.env.DNA_API_KEY,
  environment: 'production'
});

const risk = await engine.analyze('cluster_alpha');
